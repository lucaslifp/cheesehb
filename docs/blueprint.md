# **App Name**: PizzAI Ordering

## Core Features:

- Menu Display: Display a menu of available pizzas with descriptions, images, and prices.
- Order Selection: Allow users to select pizzas and specify quantities.
- Order Summary: Show a summary of the order with item quantities and total cost, including taxes.
- Sticky Total: Display the total order value. Make it sticky to the bottom.
- Contact Information: Collect minimal contact information for the order: name and phone number. Optionally address, if the shop provides delivery.
- AI Upsell Suggestion: Using provided customer information as a tool, and based on order history or preferences, AI will offer an upsell recommendation, like suggesting a specific drink, side order, or dessert. Or it can generate a custom, unique pizza based on stated preferences.
- Order processing: Process order by displaying confirmation page with estimated time. Send SMS to the phone provided during Contact Information.

## Style Guidelines:

- Primary color: Deep red (#B80F0A) to evoke the rich color of tomato sauce, and the energy of fast service.
- Background color: Light gray (#F0F0F0) to provide a clean and neutral backdrop that keeps the focus on the pizzas and other content.
- Accent color: Yellow-gold (#FFB300) to draw attention to important elements like special offers or call-to-action buttons.
- Headline font: 'Playfair', a modern serif with an elegant and fashionable feel.
- Body font: 'PT Sans', a humanist sans-serif.
- Use pizza-themed icons in a simple and easily recognizable style.
- Emphasize images in the layout, as well as the ease to add to cart.
- Subtle transitions to display added cart items, price calculation and AI offer.
